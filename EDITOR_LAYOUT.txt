KEY FEATURES:

1. SIDEBAR (Left Panel)
   - Lists ALL scripts (Persistent + Hotkey)
   - Shows "ON" badge for enabled scripts
   - Active script highlighted
   - Click to switch scripts (auto-saves)
   - Collapsible with chevron button

2. EDITOR (Main Area)
   - Full-featured CodeMirror editor
   - Syntax highlighting for Lua
   - All modern IDE features
   - Fills remaining space

3. RESIZE HANDLE
   - Drag up/down to adjust log panel size
   - Visual indicator on hover
   - Min: 100px, Max: 70vh

4. LOG PANEL (Bottom)
   - Resizable height
   - Collapsible (click chevron)
   - Shows log count badge
   - Selectable text
   - Auto-scrolls
   - Custom styled scrollbars

5. HEADER BAR
   - Script name input
   - Loop delay settings (with validation)
   - Save, Delete, Close buttons
   - Visual icons for clarity

6. VALIDATION
   - Real-time error checking
   - Visual feedback (red borders)
   - Error messages
   - Prevents invalid saves

7. KEYBOARD SHORTCUTS
   - Ctrl+S / Cmd+S: Save
   - Esc: Close
   - Ctrl+F: Find
   - Ctrl+H: Replace
   - All standard editor shortcuts

INTERACTION FLOW:

1. User opens editor by double-clicking script code
2. Editor opens with full 95vw x 95vh size
3. Sidebar shows all scripts, current one highlighted
4. User can:
   - Edit code in main area
   - Adjust loop settings in header
   - Resize log panel by dragging
   - Collapse/expand sidebar or logs
   - Switch to another script by clicking sidebar
   - Save with button or Ctrl+S
   - Close with X or Esc
5. On script switch:
   - Current script auto-saves
   - New script loads in editor
   - Sidebar updates highlight
6. On save:
   - Validation runs (if persistent script)
   - If valid: saves and closes
   - If invalid: shows error, stays open
